# Agent委托公告板｜语言约定

本仓库主要语言为中文。

## 默认规则

- 面向开发者和使用者的主文档默认使用中文。
- GitHub 首页使用中文 `README.md`。
- 英文 README 作为辅助入口，文件为 `README.en.md`。
- 新增设计文档、开发文档、联调文档优先写中文。

## 例外

以下内容保留英文：

- 代码标识符。
- JSON 字段名。
- schema 标题和协议常量。
- CLI 参数名。
- package/module 名称。
- 对外兼容矩阵中的 capability 名称。

这些名称属于跨 Agent、跨语言、跨平台的协议接口，保持英文更利于实现稳定。

## 翻译策略

如果未来需要更完整的英文文档，应新增 `*.en.md` 或英文章节，不应把中文主文档改回英文主导。
